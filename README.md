# Intranet Campeón Mundial

Sistema intranet. se implementó con el objetivo de optimizar el proceso de matricula para estudiantes de la escuela de manejo campeón mundial.

# Herramientas utilizadas:

- Reactjs
- Bootstrap
- Typescript
- Express
- Node
- Mongoose
